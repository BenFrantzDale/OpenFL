Preform has three main categories of exposure settings:

  • Base
  • Support
  • Model

The distinction is shown below:

When slicing, we tag regions with meta-data about what category it came from. This information will be used to apply exposure settings.

The algorithm that generates laser paths from a slice, uses the ‘[PrintSettings]‘ block of the material file.  At a high level a region consists of a perimeter/skin and a bulk fill region.

The algorithm parameters are as follow:

There are also the variables:

  • SliceHeight
  • Xcorrectionfactor
  • Ycorrectionfactor

Slice height is intuitively the slicer resolution, and X/Ycorrectionfactor’s are gross scale/volumetric shrinkage correction factors. The correction factors are applied to the whole model/scene prior to slicing.

Now we have a set of paths (sequence of vectors) each of which is tagged with a category (model, support, base) and a type (fill, perimeter). Based on this meta-data Preform will apply speeds and powers from the [perimeter] and [fill] blocks of the .SLP
